Nortzuk gara

Gure bezeroen gogobetetzea da gure aktiborik handiena.

 

Gure jarduera urteekin batera handitzen doa. Web teknologian hasita zenbait esparrutara hedatu gara

Negozio jardureak

Web aplikazioen garapena

  • Inplementazio malguak eta eskalagarriak
  • Estandarretan oinarritutako garapenak, hornitzaileekiko mendekotasuna saihestuz
  • Plataforma anitzeko soluzioak

Teknologia mugikorreko aplikazioak

  • Enpresen eta industriaren mugikortasunerako aplikazioetan eta estrategietan espezializatutako aholkularitza.
  • Android natibo, hibrido eta webapps aplikazioen garapena.
  • Aplikazioen diseinua eta UX/UI erabilgarritasun-kontsultoria.

Sistema txertatuak

  • Yocto Project Linuxen oinarritutako sistema pertsonalizatuak garapena.
  • Urrutiko eguneraketak dituzten soluzioak
  • Erabilerara bideratutako sistemen diseinua

Datuak

  • BigData, IoT, ...
  • Ezagutza teknologikoak hardwarean
  • Datuaen biltzea eta erabilera
  • Bussiness Inteligent soluzioak: Elasticsearch, Kibana...

Open Source ekarpenak

  • Lamassu IoT:Gauzen Interneterako (IoT) gako publikoko azpiegitura bat (PKI), oinarrizko hardwarean nahiz hodei-azpiegituran funtzionatzeko diseinatua dagoena.
  • CouchDB relax: Pitzadurarik gabeko multimatster sinkronizazioa, API http/JSON intuitibo batekin, fidagarritasunerako diseinatua.
  • Cloudant Sync - Android Datastore: Android aplikazioetarako JSON dokumentu-biltegia.
  • tusd: Go aplikazioan erreferentzia-zerbitzaria inplementatzea 'tus'-erako: Fitxategiak berriro kargatzeko protokolo irekia.
  • tus JAVA client: 'tus'-erako Java bezeroa.
  • CouchDB Java API: LightCouch CouchDBko Java bezeroaren fork aktiboa.
  • Android Json Wizard:'Android Material Json Form Wizard' zure aplikazioaren barruan inprimakietan oinarritutako lagungarriak sortzeko liburutegi bat da, json formatu jakin bat erabiliz.
  • SQL Definition Support (LGPL): SQL kontsultak Java kodearen bidez kanporatzeko laguntza-liburutegia (LGPL).
  • DB Call CDI Extension (LGPL):CDI luzapena, datu-basean biltegiratutako prozeduretara deitzeko.
