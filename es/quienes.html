Quiénes Somos

Nos dedicamos al diseño e implantación de Sistemas de Información basadas en Tecnología Web.

 

La satisfacción de nuestros clientes es nuestro mayor activo y consideramos la tecnología un medio para dar soluciones a los retos de nuestros clientes.

Indaba, como parte del grupo empresarial LKS Next, comparte la estrategia «ContinuosNEXT» consciente de que sólo desde la innovación constante es capaz de ofrecer valor a sus clientes.

A través de tecnología y soluciones de Software libre, consideramos que tenemos la capacidad de ayudar en los procesos de trasformación digital de nuestros clientes de una forma eficiente y ágil.

Ser parte del grupo LKS Next nos permite compartir experiencias, conocimiento, tecnología,... y ofrecer siempre a nuestros clientes la mejor solución posible.

Contribuciones Open Source

  • Lamassu IoT: Una infraestructura de clave pública (PKI) para el Internet de las cosas (IoT) diseñada para funcionar tanto en hardware básico como en infraestructura de nube.
  • CouchDB relax: Sincronización multimáster sin fisuras, con una API HTTP/JSON intuitiva y diseñada para la fiabilidad.
  • Cloudant Sync - Android Datastore: Un almacén de documentos basado en JSON para aplicaciones Android.
  • tusd: Implementación del servidor de referencia en Go para tus: el protocolo abierto para la carga de archivos reanudable.
  • tus JAVA client: Cliente tus para Java.
  • CouchDB Java API: Fork activo del cliente Java de LightCouch CouchDB.
  • Android Json Wizard:Android Material Json Form Wizard es una biblioteca para crear atractivos asistentes basados en formularios dentro de tu aplicación simplemente definiendo json en un formato determinado.
  • SQL Definition Support (LGPL): Biblioteca de apoyo para externalizar las consultas SQL desde el código Java (LGPL).
  • DB Call CDI Extension (LGPL):Extensión CDI para llamar a los procedimientos almacenados de la BD.
